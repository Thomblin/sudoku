# sudoku

## About

sudoku is just a little sandbox to solve sudoku puzzles.

## Usage

php run.php FILENAME

## Example

The given file shall contain the puzzle and nothing else.
It shall contain either a known number, or '#' if nothing is prefilled.


    ###56####
    1#5###2##
    2###739##
    ####1####
    #2##84639
    9782#####
    #######45
    4#2##98#7
    3####71##
